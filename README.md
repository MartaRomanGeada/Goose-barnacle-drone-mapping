# Goose-barnacle-drone-mapping
R code associated with the study on goose barnacle drone mapping  

Scripts for the study on gosse barnacle mapping with drone visible-NIR reflectance and Light detection and ranging (LiDAR) data.
Created by Marta Román (https://orcid.org/0000-0002-7082-0763)
